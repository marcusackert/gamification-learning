Level 1: Unlock the Gate

Assign a number to a variable x.

Variables are like containers. Use = to store a number in x.

Level 2: Light the Torch

Create a string variable called name.

Strings are text wrapped in quotes. Imagine writing your hero’s name on a banner.

Level 3: Collect the Treasure

Create a list with numbers 1, 2, 3.

Lists are like treasure chests. Use square brackets to hold multiple items.

Level 4: Cross the Bridge

Make a tuple named coords with two numbers.

Tuples are like locked pairs. Use parentheses to store coordinates.

Level 5: Gear Up

Python has built‑in functions to change types. Think of turning text “5” into the number 5.
